> 基于历史对话整理的股票投资 Agent 构建计划，待后续执行。

## 背景

用户希望构建一个股票智能 Agent 系统，具备：
- 实时处理市场数据
- 执行策略分析
- 提供决策支持

## 核心技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 数据获取 | finance-data-retrieval | 209 个 API，覆盖 15 类数据 |
| 分析决策 | trading-analysis | 多角色辩论框架 |
| 技术指标 | TA-Lib / 自实现 | MACD/RSI/布林带等 |

## 用户确认需求

| 需求项 | 值 |
|--------|-----|
| 数据频率 | 日级别 |
| 关注板块 | 机器人、PCB、CPO、电网、通信、医药 |
| 技术指标 | K线、MACD、RSI、布林带 |
| 目标 | 买股票（选股决策） |

## 架构设计

```
数据获取层(finance-data-retrieval) 
    → 核心扩展层(板块管理/技术指标/风险管理) 
    → 分析决策层(trading-analysis) 
    → 输出层(报告/预警/可视化)
```

## 实施路线图

| 周次 | 目标 |
|------|------|
| 第1周 | 基础框架搭建，验证插件可用性 |
| 第2周 | 技术指标计算、K线形态识别 |
| 第3周 | 风险管理、仓位计算模型 |
| 第4周 | 性能优化、生产部署 |

## 本次任务新增配置

### 策略参数
- 风险偏好：无权重偏好（均衡型）
- 最大回撤限制：10%
- 单股仓位上限：100%

### 试点计划
- 优先板块：机器人
- 首要目标：验证技术流程可用性
