## 🔧 4. backtesting-frameworks 代码模板

该技能提供 Python 回测框架代码模板，包含：

### 核心组件
- **Order/Fill/Position** - 订单、成交、持仓管理
- **Portfolio** - 账户权益计算
- **Strategy** - 策略抽象基类
- **ExecutionModel** - 执行模型（含滑点、佣金）
- **Backtester** - 回测引擎

### 避免偏差的机制
| 偏差类型 | 解决方案 |
|----------|----------|
| Look-ahead | Point-in-time 数据 |
| Survivorship | 包含退市股票 |
| Overfitting | Out-of-sample 测试 |
| Transaction | 真实成本模型 |

### 使用示例（MA双均线策略）
```python
class MAStrategy(Strategy):
    def on_bar(self, timestamp, data):
        # MA20 上穿 MA60 买入
        if data['MA20'] > data['MA60'] and data['MA20_prev'] <= data['MA60_prev']:
            return [Order(symbol='300750.SZ', side=BUY, quantity=100)]
```
