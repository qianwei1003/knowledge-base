## 🎯 核心技能详解

### 1️⃣ neodata-financial-search（数据层）

**能力验证**: ⭐⭐⭐⭐⭐

**实测功能**:
- ✅ 个股实时行情（价格/涨跌幅/市值）
- ✅ 历史行情数据（OHLCV）
- ✅ 财报数据（利润表/资产负债表/现金流量表）
- ✅ 估值分析（PE/PB/PS/ROE/毛利率）
- ✅ 资金流向（主力/散户净流入）
- ✅ 基金数据（净值/回报率）
- ✅ 宏观数据（Shibor/LPR/国债收益率）
- ✅ 板块资讯

**响应速度**: 2-3秒  
**数据质量**: 实时（T+0）

**调用方式**:
```bash
python "C:\Users\60597\.codebuddy\plugins\marketplaces\cb_teams_marketplace\plugins\finance-data\skills\neodata-financial-search\scripts\query.py" --query "查询内容"
```

---

### 2️⃣ trading-analysis（分析层）

**能力验证**: ⭐⭐⭐⭐⭐

**架构**: 11个Agent多角色协作
- Phase 1: 4个分析师并行（市场/基本面/新闻/情绪）
- Phase 2: 多头vs空头辩论
- Phase 3: 交易员决策
- Phase 4: 3个风险分析师并行
- Phase 5: 整合报告

**输出内容**:
- ✅ BUY/SELL/HOLD 明确建议
- ✅ 信心水平（高/中/低）
- ✅ 建议仓位比例
- ✅ 入场价位区间
- ✅ 目标价位
- ✅ 止损价位
- ✅ 核心理由（3-5句话）

**触发方式**: 自然语言提问
```
"分析一下300750能不能买入"
"给我一份宁德时代的投资分析报告"
"评估一下300750的投资价值"
```

**适用场景**: 重大投资决策、大额资金操作

---

### 3️⃣ idea-generation（选股层）

**能力验证**: ⭐⭐⭐⭐

**功能**: 系统化股票筛选
- 价值筛选（低PE/PB、高股息）
- 成长筛选（高增长、高ROE）
- 质量筛选（稳健盈利、低负债）
- 事件驱动（ catalyst 筛选）

**输出内容**:
- 5-10个投资idea
- 一页纸摘要/股
- 关键指标vs同行对比
- 投资逻辑（3-5点）
- 关键风险
- 下一步建议

**触发方式**:
```
"帮我筛选一些价值股"
"找找电力设备板块的投资机会"
"筛选PE低于20倍且ROE>15%的股票"
```

---

### 4️⃣ risk-metrics-calculation（风控层）

**能力验证**: ⭐⭐⭐⭐

**功能**: 计算投资组合风险指标
- VaR (Value at Risk)
- CVaR/ES (Expected Shortfall)
- 夏普比率 (Sharpe Ratio)
- 索提诺比率 (Sortino Ratio)
- 最大回撤 (Max Drawdown)
- Beta系数
- 下行偏差

**代码模板**: 提供Python实现

**触发方式**:
```
"计算300750的风险指标"
"如果买入10万宁德时代，最坏情况亏多少"
"评估持有300750的VaR"
```

---

### 5️⃣ backtesting-frameworks（验证层）

**能力验证**: ⭐⭐⭐⭐

**功能**: 交易策略回测框架
- 事件驱动回测引擎
- 向量化回测
- Walk-Forward分析
- 蒙特卡洛模拟

**避免偏差**:
- Look-ahead bias
- Survivorship bias
- Overfitting
- Transaction costs

**代码模板**: Python实现

**触发方式**:
```
"回测MA20上穿MA60策略，标的300750"
"验证双均线策略在宁德时代的表现"
"测试突破买入策略的胜率"
```

## 🎯 核心结论

### 技能生态完整性: ⭐⭐⭐⭐⭐

市场上已有的股票技能生态非常完善：
- **数据层**: neodata-financial-search (209个API)
- **分析层**: trading-analysis (11个Agent)
- **选股层**: idea-generation (系统化筛选)
- **风控层**: risk-metrics-calculation (VaR/Sharpe)
- **验证层**: backtesting-frameworks (回测框架)

### 与自建技能对比

| 维度 | 自建 buy-signal-analyzer | 官方 trading-analysis |
|------|-------------------------|----------------------|
| 角色 | 单一Agent | 11个专业Agent |
| 流程 | 直接计算评分 | 5阶段工作流 |
| 输出 | 0-100评分 | 完整报告+BUY/SELL/HOLD |
| 质量 | 中等 | 专业级 |
| 维护 | 需自行维护 | 官方维护 |

**结论**: 官方 `trading-analysis` 已完全覆盖自建技能场景，且质量更高。

### 推荐使用方案

| 用户类型 | 推荐技能组合 |
|----------|-------------|
| 散户投资者 | neodata + trading-analysis |
| 价值投资者 | neodata + idea-generation + trading-analysis |
| 量化投资者 | neodata + backtesting + risk-metrics |
| 机构投资者 | 全套技能组合 |
