# 股票技能测试清单

> 用途: 按此清单逐一测试技能，熟悉使用方法和输出效果
> 预计时间: 每技能 5-10 分钟

---

## 测试环境准备

1. 打开 CodeBuddy 对话框
2. 确保 finance-data 插件已连接
3. 准备一只熟悉的股票作为测试标的（建议：宁德时代、贵州茅台、比亚迪）

| 3.3 | "比亚迪值得投资吗？深度分析" | 投资建议+仓位建议+目标价 |
| 3.4 | "帮我分析这个股票：代码300750" | 代码识别分析 |

**预期报告结构**（测试时核对）：
- [ ] 最终建议：BUY/SELL/HOLD
- [ ] 信心水平：高/中/低

- [ ] 建议仓位：X%
- [ ] 技术面分析（趋势、支撑压力位）
- [ ] 基本面分析（盈利能力、估值）
- [ ] 情绪面分析（资金流向）
- [ ] 多头vs空头辩论

**测试通过标准**：
- [ ] 报告包含以上所有章节
- [ ] 给出明确的BUY/SELL/HOLD建议
- [ ] 包含具体的价格建议

## 测试建议顺序

### 第1天：数据查询（30分钟）
- neodata-financial-search（必测，测试3-5个问题）
- finance-data-retrieval（选测，测试历史数据）

### 第2天：投资分析（60分钟）
- trading-analysis（必测，选1只熟悉的股票深度分析）
- idea-generation（必测，尝试2-3种筛选条件）

### 第3天：量化工具（60分钟）
- backtesting-frameworks（必测，回测1个简单策略）
- risk-metrics-calculation（必测，计算1只股票的风险）

### 第4天：专业研究（60分钟）
- earnings-analysis（选测）
- sector-overview（选测）
- catalyst-calendar（选测）

### 第5天：综合应用（30分钟）
- 尝试组合使用多个技能完成一个完整的投资决策流程
