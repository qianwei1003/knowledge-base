> 股票投资 Agent 系统的策略参数配置

## 风险参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **风险偏好** | 无权重偏好 | 均衡型，不偏向激进或保守 |
| **最大回撤限制** | 10% | 触发止损预警阈值 |
| **单股仓位上限** | 100% | 单只股票最大仓位比例 |

## 仓位管理规则

```yaml
position_rules:
  max_single_position: 1.0      # 单股最大仓位 100%
  max_sector_exposure: 0.3      # 单板块最大暴露 30%
  stop_loss: 0.10                # 止损线 10%
  take_profit: 0.20              # 止盈线 20%
  max_total_exposure: 1.0        # 最大总仓位 100%
```
