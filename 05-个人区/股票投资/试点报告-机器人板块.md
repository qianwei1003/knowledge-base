## finance-data 插件 Token 配置

### 问题诊断

API调用返回 `code: 40101, msg: "您的token不对，请确认"`

### 正确的配置方式

根据腾讯云官方教程，finance-data 插件使用 **Financial Datasets** API：

#### 1. 申请 API 密钥

访问：https://www.financialdatasets.ai/

> ⚠️ 注意：如果网络无法访问该申请页面，则无法使用该功能。

#### 2. 本地 MCP 服务器配置

该插件不是 CodeBuddy 内置 API，而是需要本地运行 MCP 服务器：

```bash
# 1. 克隆代码仓库
git clone https://github.com/financial-datasets/mcp-server

# 2. 安装 uv 工具
# Windows:
irm https://astral.sh/uv/install.ps1 | iex

# 3. 创建虚拟环境并安装
uv venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
uv add "mcpcli" httpx

# 4. 设置 API 密钥
cp .env.example .env
# 编辑 .env 文件，填入: FINANCIAL_DATASETS_API_KEY=你的密钥

# 5. 配置到 CodeBuddy (settings.json)
{
  "mcpServers": {
    "financial-datasets": {
      "command": "uv",
      "args": ["--directory", "D:/path/to/mcp-server/financial-datasets/mcp-server", "run", "server.py"]
    }
  }
}
```

#### 3. 重要说明

| 项目 | 说明 |
|------|------|
| 密钥来源 | financialdatasets.ai（需申请） |
| 运行方式 | 本地 MCP 服务器 |
| 数据范围 | 财务报表、实时股价、市场新闻 |
| 适用场景 | 投资分析、股票筛选 |
