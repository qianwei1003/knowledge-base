## 使用建议

### AI 编程时的规范调用

当 AI 编写代码时，可引用对应规范：

```markdown
请遵循 Airbnb JavaScript 规范，特别是：
- 优先使用 const
- 使用 === 而非 ==
- 使用 ES6 模块
```

### 规范冲突时的优先级

1. **项目级规范** > 公司规范 > 通用规范
2. **团队共识** > 个人偏好
3. **最新规范** > 旧版规范
