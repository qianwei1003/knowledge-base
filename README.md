> 基于 Obsidian 的模块化知识管理，支持 AI 协作产出。

## 🏗️ 架构

```
知识库/
├── 00-Index/          # 总索引和管理
├── 01-工作区/         # 日常工作、会议、AI对话
├── 02-技术规范/        # codebuddy-craft 专业规范
├── 03-项目库/         # 各项目文档
├── 04-学习研究/       # 学习笔记
├── 05-个人区/         # 个人管理
└── 06-AI协作区/       # AI 协作专用
```

## 🤖 AI 协作

本知识库支持**多 AI 协作**：
- Claude Code
- GitHub Copilot
- 其他 AI 工具

详见 [[00-Index/AI协作规范]]

## 🏷️ 标签体系

使用统一标签管理内容：
- `#来源/Claude` - Claude Code 产出
- `#来源/User` - 用户记录
- `#技术/xxx` - 技术分类
- `#工作区/xxx` - 工作分类

详见 [[00-Index/标签体系]]

## 🚀 快速开始

1. **克隆仓库**
```bash
git clone <仓库地址>
```

2. **安装 Obsidian**（如未安装）
- 下载: https://obsidian.md

3. **打开知识库**
- Obsidian → 打开文件夹 → 选择本目录

4. **查看使用指南**
- [[00-Index/知识库使用指南]]
