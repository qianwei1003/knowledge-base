# 知识库使用指南

## 🎯 目的

本知识库作为个人/团队的**中央信息枢纽**，所有重要知识、沟通记录、AI 协作产出都统一记录在这里。

## 📁 目录结构

```
知识库/
├── 00-Index/          # 总索引和管理
├── 01-工作区/         # 日常工作
│   ├── 00-Inbox/      # ⭐ 收集箱（新内容先放这里）
│   ├── 01-会议记录/
│   ├── 02-AI对话/
│   ├── 03-工作日志/
│   └── 04-决策记录/
├── 02-技术规范/        # codebuddy-craft 专业规范
├── 03-项目库/         # 各项目文档
├── 04-学习研究/       # 学习笔记
├── 05-个人区/         # 个人管理
└── 06-AI协作区/       # AI 协作专用
```

## ✍️ 写入规范

### 文件命名
```
日期-主题.md                    # 例：2026-04-18-会议记录.md
YYYY-MM-DD-主题-类型.md         # 例：2026-04-18-Claude-代码审查.md
```

### 元信息头
```markdown
---
created: 2026-04-18
modified: 2026-04-18
source: #来源/Claude
tags: [#技术/前端, #工作区/对话]
---

# 内容标题
```

### 来源标识（必须）
| 标识 | 说明 |
|------|------|
| `#来源/Claude` | Claude Code 产出 |
| `#来源/Copilot` | Copilot 产出 |
| `#来源/User` | 用户手动记录 |

## 🔄 知识流转

```
收集(Inbox) → 整理(分类) → 关联(双链) → 输出(产出)
     ↓              ↓            ↓           ↓
  临时笔记      移动到正确    添加 [[链接]]   放入对应模块
               模块目录       完善 MOC
```

## 🛠️ 常用操作

### 添加新笔记
1. 放入 `01-工作区/00-Inbox/` 
2. 根据内容移动到对应模块
3. 更新相关 MOC 索引

### 与 Claude Code 协作
1. 在项目中开始对话
2. 重要产出要求 Claude 写入知识库
3. Claude 会自动添加来源标识

### 搜索和导航
- 使用 Obsidian 搜索 `Ctrl+O`
- 通过 MOC 地图浏览
- 使用标签筛选 `#标签名`
