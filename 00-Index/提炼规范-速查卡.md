## 📊 提炼层级

| 层级 | 目标 | 耗时 | 产出位置 |
|------|------|------|---------|
| **L1 初提炼** | 提取核心要点 | 15-30min | `待提炼/` |
| **L2 精提炼** | 整合多源，消除冲突 | 30-60min | `06-AI协作区/提炼知识/` |
| **L3 规则化** | 转换为可执行规则 | 60-120min | `rules/` |

---

## ✅ 各阶段检查清单

### L1 初提炼检查清单

```
□ frontmatter 信息完整（6个字段）
□ 核心要点速查已添加（5-10条）
□ 关键段落已标注来源（> 来源：xxx）
□ 冗余内容已删除
□ 提炼状态已更新
```

### L2 精提炼检查清单

```
□ 相同主题文档已归类
□ 多来源冲突已标注（多源对比表）
□ 统一规则已形成
□ 使用标准模板
□ 来源追溯表已添加
□ 检查清单已编写
```

### L3 规则化检查清单

```
□ 规则文件位置正确
□ 无重复规则
□ 每条规则有示例（正例+反例）
□ 每条规则可检查
□ 规则来源已标注
□ 通过可执行性验证
```

## ⭐ 质量星级评定

| 星级 | 标准 | 处理建议 |
|------|------|---------|
| ⭐⭐⭐⭐⭐ | 权威、完整、可直接转化 | 优先提炼 |
| ⭐⭐⭐⭐ | 较完整、有权威性 | 正常提炼 |
| ⭐⭐⭐ | 一般、需整理 | 选择性提炼 |
| ⭐⭐ | 内容较少 | 合并或搁置 |
| ⭐ | 质量差 | 删除 |
