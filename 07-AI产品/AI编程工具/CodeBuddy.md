> 腾讯云推出的新一代 AI 代码编辑器，提供代码补全、生成、诊断、技术问答和性能优化等功能。

## 产品概述

| 项目 | 内容 |
|------|------|
| 名称 | CodeBuddy |
| 厂商 | 腾讯云 |
| 类型 | AI 编程助手 |
| 定位 | 深度集成腾讯云生态的智能编程工具 |

## 核心能力

| 能力 | 说明 |
|------|------|
| 代码补全 | 支持 Java/Python/Go/JS 等主流语言 |
| 代码生成 | 自然语言描述 → 自动生成代码 |
| 错误诊断 | 代码错误检测与修复建议 |
| 技术问答 | 编程问题解答 |
| 性能优化 | 代码性能分析与建议 |
| 云端协同 | 一键操作腾讯云资源 |

## 产品形态

| 形态 | 说明 |
|------|------|
| VSCode 插件 | 主要形态，深度集成 VSCode |
| IDE 插件 | 支持多种 IDE |
| Cloud Studio | 云端开发环境 |
| CLI 工具 | 命令行工具 |

## 插件市场

CodeBuddy 支持丰富的插件扩展：

| 插件 | 功能 |
|------|------|
| finance-data | 金融数据检索（209个API） |
| trading-analysis | 多角色辩论投资分析 |
| pdf | PDF 处理 |
| xlsx | Excel 处理 |
| docx | Word 处理 |
| playwright | E2E 测试 |
