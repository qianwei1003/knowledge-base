> 腾讯电脑管家推出的桌面AI助手，基于 OpenClaw 开源框架开发，微信远程操控电脑。

## 产品概述

| 项目 | 内容 |
|------|------|
| 名称 | QClaw（小龙虾） |
| 出品方 | 腾讯电脑管家 |
| 基础框架 | OpenClaw（开源） |
| 内置模型 | Kimi-2.5 |
| 官方网站 | https://qclaw.qq.com/ |

## 核心能力

| 能力 | 说明 |
|------|------|
| 微信直连 | 绑定微信，远程操控电脑 |
| 本地部署 | 数据本地处理，保护隐私 |
| Skills 生态 | 5000+ 技能，支持 ClawHub/GitHub |
| 持续记忆 | 记住用户偏好，越用越懂你 |
| 文件操控 | 直接操作本地文件、浏览器、邮件 |
| 自动开发 | 创建项目、提交 GitHub、生成文档 |

## 使用场景

| 场景 | 功能 |
|------|------|
| 远程操作 | 远程打开/编辑文件、操控网页 |
| 资料整理 | 自动归类文件、提炼摘要 |
| 智能提醒 | 定时推送、穿衣出行建议 |
| 编程开发 | 自动开发、代码生成 |
| 学术研究 | 论文检索、文献整理 |
| 文章撰写 | 辅助写作、学习风格 |

## 产品特点

1. **零门槛上手**：扫码绑定微信，三步完成配置
2. **跨平台支持**：Mac 和 Windows 全面覆盖
3. **微信直连**：随时随地发指令
4. **本地化执行**：直接操控本地文件和应用
5. **AI 进化**：持续学习用户习惯
6. **隐私优先**：数据保留在本地设备

## 支持平台

- macOS (Apple 芯片)
- macOS (Intel 芯片)
- Windows
